# DBMS Lab 5 – INNER JOIN Queries

## Objective
To understand and implement INNER JOIN operations to retrieve data from multiple related tables using primary–foreign key relationships.

## Tables Used
- Student
- Enrollment
- Course
- Faculty
- Department

## Concepts Practiced
- INNER JOIN
- Multi-table JOIN
- Table Aliases
- WHERE clause (Filtering)
- ORDER BY (Sorting)
- GROUP BY
- Aggregate Functions (COUNT, MAX)

## Lab Work
- Joined Student and Enrollment tables.
- Joined Course and Faculty tables.
- Performed triple JOIN (Student–Enrollment–Course).
- Executed department-based queries.
- Applied filtering and sorting.
- Performed analytical queries like counting students and finding maximum grades.

## Conclusion
This lab improved understanding of relational database concepts and practical use of INNER JOIN for retrieving meaningful data from multiple tables.
